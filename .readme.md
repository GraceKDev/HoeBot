## Hobot Discord Bot

Hobot is a modular Discord bot built with discord.py, designed to enhance your server with interactive commands, user input modals, automated tasks, and custom utilities.

### Features
- **Custom Commands:** Fun and useful commands like `hug`, `slap`, and `joined` to engage your community.
- **Modals for User Input:** Collect information from users, such as completed games and birthdays, using Discord modals.
- **Automated Tasks:** Background tasks like birthday checks to automate server management.
- **Paginated Views:** Display lists (e.g., birthdays) in a user-friendly, paginated format.
- **Utility Functions:** Centralized helpers for command handling and database operations.
- **Testing Suite:** Automated tests to ensure bot reliability and correctness.

### Project Structure
- `src/` — Main source code, including commands, modals, tasks, utilities, and views
- `tests/` — Unit tests for bot commands and features


### Getting Started
1. Install dependencies with Poetry or pip.
2. Configure your Discord bot token and environment variables.
3. Run the bot from the `src/main.py` file.

---
Hobot is designed for easy extension and customization. Contribute your own commands, modals, or tasks to make your server unique!
